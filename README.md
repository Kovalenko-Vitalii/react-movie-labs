# Movie App
This is a simple movie web app that use TMDB API.  
User can see popular movies, write comments, see movie details, and check actors.
## Features
- Show popular movies
- Movie details page (rating, overview, genres, actors, recommendations, similar)
- Comments for each movie
- Actor page with movies list
- Basic filtering, searching and sorting
- Pagination
- "Watchlist" and "Favourites" pages for saving movies
## Technologies
- React
- React Router
- React Query
- TMDB API
## How to Run
1. Clone repository:
   git clone https://github.com/Kovalenko-Vitalii/react-movie-labs.git
2. Install package:
   npm install
3. Create `.env` file and put:
   VITE_TMDB_API_KEY=your_api_key
4. Start the app:
   npm run dev
## API
Data from:
https://www.themoviedb.org/
